# Odoo FedEx App

Aplikacja do integracji Odoo z FedEx API - generowanie etykiet, sprawdzanie cen, walidacja adresów.

## Funkcje

- Integracja z Odoo (pobieranie zamówień)
- Integracja z FedEx API (generowanie etykiet, sprawdzanie cen)
- Automatyczne planowanie odbioru (USE_SCHEDULED_PICKUP)
- Walidacja adresów
- Obsługa wielu formatów etykiet (PDF, ZPL)
- Automatyczne aktualizacje z GitHub Releases

## Instalacja

1. Pobierz najnowszą wersję z [Releases](https://github.com/KreativiaBird/odoo-fedex-app/releases)
2. Uruchom `OdooFedExApp.exe`
3. Skonfiguruj połączenia z Odoo i FedEx w zakładce Ustawienia

## Wymagania

- Windows 10/11
- Połączenie z internetem
- Konto Odoo z dostępem API
- Konto FedEx z kluczami API
